Hello! My name is Christabel and I am currently a university student studying a Bachelor's Degree in Digital Arts. Here I will be writing about my favourite topic within my Applied Software Practices study-unit; 2D Sprite Animation & 3D Scuplting.




Basic Information


This topic mainly served as an introductory unit which introduced us – the students – to two very important software programs, these being Unity and Blender. Unity is a game engine which enables users to create interactive games and apps whilst Blender is 3D graphics software which could be used for numerous purposes such as for animation and computer games.

For more information about Unity and Blender one could visit their official website from the following links:

  • Go to Unity
  • Go to Blender


Practical Assignments


As part of our requirements for this topic we had to submit two deliverables. One of these deliverables required the production of an animation of a 2D character of our choice. The 2D character was to be created through Piskel (an online editor for pixel art) doing multiple actions which was then to be exported as a sprite sheet. This sprite sheet is then used to create the animation through Unity.


2D character animation developed from a sprite sheet which contained my alien character.


As for the second deliverable we had to sculpt a 3D still life of a fruit bowl containing a couple of fruits by using the Blender software and present it as a turntable animation.
